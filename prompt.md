Crie uma landing page onepage profissional (nível SaaS moderno) para um desenvolvedor web freelancer.

## OBJETIVO

A página deve transmitir autoridade, clareza e profissionalismo, com foco em conversão de clientes que desejam contratar serviços de desenvolvimento web (sites e sistemas).

## ESTILO VISUAL

* Tema DARK moderno (fundo escuro elegante)
* Estilo minimalista, clean e sofisticado
* Inspirado em landing pages SaaS modernas
* Tipografia bem definida (ex: Inter, Poppins ou similar)
* Uso de gradientes sutis (roxo, azul, ou verde tecnológico)
* Alto contraste e ótima legibilidade

## TECNOLOGIAS

* HTML5
* CSS3 (ou TailwindCSS se preferir)
* JavaScript puro (sem frameworks pesados)

## ANIMAÇÕES (IMPORTANTE)

Adicionar animações modernas com JavaScript:

* Scroll reveal (elementos surgindo suavemente)
* Fade-in + slide (horizontal e vertical)
* Hover effects em botões e cards
* Microinterações suaves
* Navbar com efeito ao rolar (blur ou mudança de cor)
* Animação leve no Hero (ex: partículas, gradient animation ou typing effect)

Pode usar bibliotecas leves como:

* AOS (Animate On Scroll)
* GSAP (se necessário, mas manter leve)

## ESTRUTURA DA PÁGINA

### 1. HERO (primeira dobra)

* Headline forte:
  "Desenvolvedor Web & Criador de Soluções Digitais"
* Subheadline:
  "Transformo ideias em sistemas e sites rápidos, funcionais e prontos para crescer."
* Botão CTA: "Falar comigo"
* Elemento visual (mockup, código ou animação)

### 2. SOBRE / O QUE FAÇO

* Texto curto explicando:

  * Desenvolvimento de sites
  * Sistemas web personalizados
  * APIs e integrações
  * Automações

### 3. TECNOLOGIAS

Exibir em grid com ícones:

* Node.js
* PHP (CodeIgniter 4)
* MongoDB
* MySQL
* Docker
* AWS

### 4. PROJETOS

Criar 3 cards modernos com:

* Nome do projeto
* Descrição
* Tags de tecnologia
* Efeito hover elegante

### 5. DIFERENCIAIS

Lista com ícones:

* Visão de produto
* Código limpo e escalável
* Experiência com deploy real
* Foco em solução, não só código

### 6. COMO EU TRABALHO

Etapas:

1. Entendimento
2. Planejamento
3. Desenvolvimento
4. Entrega

### 7. CTA FINAL

Frase:
"Tem uma ideia ou projeto? Vamos tirar isso do papel."

Botão destacado

### 8. RODAPÉ

* Nome
* GitHub
* Email
* Direitos reservados

## RESPONSIVIDADE

* Totalmente responsivo (mobile-first)
* Layout adaptado para celular, tablet e desktop

## QUALIDADE DO CÓDIGO

* Código organizado e comentado
* Separação clara de HTML, CSS e JS
* Boas práticas de desenvolvimento
* Performance otimizada

## DIFERENCIAL

A página NÃO deve parecer um template genérico.
Deve parecer um produto profissional pronto para vender serviços.

Caprichar no design, espaçamento, animações e experiência do usuário.

Gere tudo em arquivos separados:

* index.html
* style.css
* script.js
